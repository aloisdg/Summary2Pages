# Summary2Pages

Generate folders and files from your [GitBook](https://github.com/GitbookIO/gitbook)'s [SUMMARY.md](https://github.com/GitbookIO/gitbook/blob/master/docs/SUMMARY.md) file.

It is a work in progress.
